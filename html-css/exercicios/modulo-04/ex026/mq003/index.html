Mude a orientação do seu dispositivo

Esse modo só é bom usar se for projetos pequenos, mas não é tão bom usar.

Media query é a mesma coisa que Media Type e Media Features. Posso usar os dois ao mesmo tempo [media="screen and (orientation: portrait)"]

Media Type é tipos de mídias.

Media Features é características de mídias. escrita em parênteses.
