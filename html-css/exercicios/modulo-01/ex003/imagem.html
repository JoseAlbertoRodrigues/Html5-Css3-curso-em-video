Baixar o GIMP para tratamento de imagens

Formatos de arquivos para utilizar imagens, para a WEB, formatos jpeg e png.

jpeg => o tamanho dele é mais maneiro para carregar em web, ou seja, ele tem alto compactação
e o png => ele não tem a alto compactação, mais ele também é compactado, ou seja tamanho leve, e permite a transparência.
quero colocar uma foto no fundo da minha tela, melhor formato é o jpeg..
quero uma imagem flutuando em cima e o fundo transparênte, só vai servi o png.
